# Topic-modeling-clean-energy-news
This project was developed as my undergraduate thesis at Universitas Gadjah Mada. The objective is to analyze over 20,000 clean energy news articles using Latent Dirichlet Allocation (LDA). The dataset used in this project is the Cleantech Media Dataset from Kaggle, which consists of more than 20,000 media articles related to cleantech published between 2022 and 2024. This project compares two text representation strategies before LDA, namely TF-IDF as a statistical representation and KeyBERT as a semantic embedding-based representation. Model evaluation is conducted using coherence C_v and topic diversity. The results show that the KeyBERT-based representation produces the best topic quality at K=6, with a coherence C_v score of 0.6558 and a topic diversity score of 0.9333. Meanwhile, the best TF-IDF-based model is obtained at K=40, with a coherence C_v score of 0.6043 and a topic diversity score of 0.7050. Therefore, the KeyBERT-based representation is able to produce topics that are more concise, coherent, diverse, and easier to interpret compared to the TF-IDF representation.
